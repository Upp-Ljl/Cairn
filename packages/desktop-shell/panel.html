    Cairn — 一句话

    Cairn 是你的 AI 编程工人组合管理面。 agent（Claude Code / Cursor / Codex / Aider 等）是真正写代码的引擎； Cairn 是仪表盘 + 方向盘 + 急刹车，让你能放心走开、回来看进度、必要时拉回方向。

    5 个模块

    ① State Strip：项目跑得怎样、Whole 是啥、最近 24h 战况
    ② Mentor：状态 + Mentor 建议 / Agent 自荐 / 你的备忘（每条可派给 session）
    ③ Steer：选个 session 直接喊一句
    ④ Sessions：每个 agent session 的工作脉络 (点进去看 timeline)
    ⑤ Safety：一键回退到某个 checkpoint

    键盘快捷键

    j / k 上下浏览 activity； / 聚焦 steer 输入； Enter 发送； Esc 回上一层（在 projects 列表时关闭面板）； ? 打开这页帮助

    Mentor 边界

    Cairn 内置 Mentor 是引导员，不是决策者。战略决策（"下一个 feature 是啥"）走外部 coding agent；Mentor 只做监督 + 引导 + 辅助提效。

    🛤 Lanes — 连续迭代（"走开就行"）

    在 Mentor 建议清单里勾选 N 个 agent 自荐项 → 点 "Authorize N as lane" → Cairn 把它们排成 lane，逐个跑。每个 task 跑到 WAITING_REVIEW，Cairn 自动停下来 + Mentor 给你一句通知 + 等你点 Advance。永远不替你 accept（PRODUCT.md §1.3 #4a 守住）。这就是"走开就行"——但你回来才是决策者。
