## Privacy / secret safety

The dogfood asserts and the smokes already enforce — but to be
explicit:

- The MiniMax API key NEVER appears in any `console.log` line. The
  `describeProvider` surface only echoes `provider`, `model`, and
  `base_url_host`.
- The LLM payload (when the call succeeds) does NOT contain
  `agent_id`, `session_id`, `cwd`, `transcript`, raw report bodies,
  or capability tags. The privacy contract lives in
  `goal-interpretation.cjs::buildCompactState` (Phase 2 boundary)
  and `pre-pr-gate.cjs::llmRewrite` user message
  (only `status` / `rule_log` / `checklist` / `risks` / `evidence`).
- Smoke `smoke-goal-interpretation.mjs` injects a `__SMOKE_POISON__`
  marker into 12+ sensitive fields and asserts none reach the
  outbound payload (98/98 PASS today).
- `.cairn-poc3-keys/` is in `.gitignore:37`; no key file enters git.

## Positioning audit

This round adds three new write surfaces (goal field on registry,
worker-reports JSONL, cached interpretation/gate result in memory)
plus an outbound LLM call. The audit checklist:

- [x] No new SQLite schema / migration
- [x] No new MCP tool
- [x] No new npm dep (Node 18+ / Electron 28+ built-in `fetch`)
- [x] LLM is **advisory**: never recommends agent actions, never
      claims completion without evidence (system prompts assert this
      explicitly; smoke verifies)
- [x] LLM cannot change the Pre-PR Gate `status` (smoke asserts a
      hostile attempt to flip status is ignored)
- [x] LLM never receives raw transcripts / prompts / tool args /
      stdout / stderr / agent_id / session_id / cwd / capabilities /
      api keys (smoke poisons all 12+ sensitive fields and asserts
      none reach the payload)
- [x] Cairn does not write code (verified by grep on every prod file)
- [x] Cairn does not run tests, install hooks, or block git
- [x] Pre-PR Gate is advisory (status determined by deterministic
      rules; LLM rewrites tone only)
- [x] Read-only on `cairn.db`, `~/.claude`, `~/.codex` (mutation grep
      ≤ 1 dev-flag)
- [x] `.cairn-poc3-keys/` gitignored; key never logged
